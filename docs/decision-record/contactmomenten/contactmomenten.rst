Contactmomenten
===============

De belangrijkste functionaliteit van KISS is het registreren van de Contactmomenten: verslagje of weergaves van het contact van een burger/ondernemer met het KCC. 

Bij de start van de ontwikkeling van KISS was hiervoor alleen de VNG Contactmomenten API beschikbaar zoals deze was geïmplementeerd in Open Klant v0.5.0-pre. 

Inmiddels is Open Klant 2.x beschikbaar, waarin de Klantinteracties API is geïmplementeerd. Het object binnen deze API waarin de belangrijkste informatie van een Contactmoment wordt opgeslagen heet daar `klantcontact`. De term Contactmoment duidt binnen KISS nu op de schermen waarmee een KCM het `klantcontact` vastlegt. 

KISS ondersteund alleen nog een aangepaste versie van de Contactmomenten API t.b.v. de koppeling met de e-Suite.
